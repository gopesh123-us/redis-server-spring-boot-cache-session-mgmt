# Redis Server for Session Management, through serialization and caching with Spring Boot
This simple app demonstrates the power of Redis server working along with Spring Boot. Here Redis server is working as a in-memory database for fast real-time data processing for managing user session and also Redis server serving cached data that is accessed frequently - using serialization in the backend. 
The application is dockerized to easy deployment on any Docker container.
