Welcome to Redis Server Application with Spring Boot!

Redis Server being a fast in-memory database, this simple application demonstrates the power of Redis Server for caching data and for managing user session data using serialization and deserialization. The application includes docker-compose and Dockerfile for easy install and deployment.

Swagger API view

Technologies used

  • Spring Boot
  • REST APIs only
  • Redis Server
  • Ubuntu (WSL)
  • Swagger UI
  • User session management using serialization
  • Caching using serialization
  • Eclipse
  • Git
  • Dockerfile
  • docker-compose
